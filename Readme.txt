Readme

Langkah utama untuk menyiapkan Sistem Portal Berita menggunakan dasar PHP MySQL. Untuk menajalankan proyek khusus ini, PHP Versi 5.6.3 atau 7.4.12 dibutuhkan karena sudah teruji dengan baik pada keduanya. Jadi, jika pengguna dengan versi PHP terbaru (lebih dari 7.4.12) mungkin menghadapi berbagai masalah saat mengoperasikannya. Solusinya anda perlu menurukan versi php anda saat ini.

# Cara menjalakannya

1. Setelah Anda selesai mengunduh proyek, ekstrak file proyek dan buka direktori XAMPP/Laragon Anda.
2. Di sana Anda akan menemukan folder bernama "htdocs/www".
3. Di dalam folder "htdocs/www", Paste folder proyek (bukan yang .zip, tetapi yang diekstrak).
4. Buka browser default anda; kami menyarankan menggunakan Google Chrome, Mozilla Firefox atau Microsoft Edge.
5. Lalu, buka URL “http://localhost/phpmyadmin“.
6. Buat Databse dengan nama "livenews".
7. Klik tab “Import” dan pilih berkas basis data (.sql) yang tersedia di bawah folder dengan nama “livenews”.
8. Setelah menyiapkan semua ini, buka URL “http://localhost/winnicode1/index.php“
9. Semua detail login tersedia di dalam folder proyek, periksa dan masukkan untuk menggunakannya.
